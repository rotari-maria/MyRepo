Acest program realizează o animație a unei mașinuțe.

La început este afișat un cronometru de start. După semnalul de start, mașinuța începe să se deplaseze pe traseu către la indicatorul STOP.
Când ajunge la indicator, mașinuța se oprește pentru un scurt timp, după care își schimbă direcția și se întoarce pe același traseu până la poziția inițială.
Animația este creată prin afișarea succesivă a mai multor cadre, care creează impresia de mișcare a mașinuței.
